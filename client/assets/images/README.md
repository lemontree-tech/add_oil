# Background Image

Place your Chinese bamboo/ink painting background image here.

## Recommended Sources:

1. **Pexels**: https://www.pexels.com/search/chinese%20ink%20painting/
   - Free, high quality
   - Search: "chinese ink painting", "bamboo", "minimalist chinese art"

2. **Pixabay**: https://pixabay.com/images/search/chinese-ink-painting/
   - Free images
   - Good selection of traditional art

3. **Unsplash**: https://unsplash.com/s/photos/chinese-ink-painting
   - High resolution
   - Modern minimalist style

## Image Requirements:

- **Format**: JPG or PNG
- **Size**: At least 1080x1920 (portrait) or 1920x1080 (landscape)
- **Style**: Subtle, elegant, peaceful
- **Opacity**: Will be set to 15% in code, so image can be more vibrant

## Suggested Search Terms:

- "chinese ink painting bamboo"
- "minimalist chinese art"
- "traditional chinese painting"
- "bamboo forest minimalist"
- "ink wash painting"

## File Name:

Name the file: `bamboo_background.jpg` (or `.png`)

Then update the code in `home_screen.dart` to use the image instead of the gradient.
